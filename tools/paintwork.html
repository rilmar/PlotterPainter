Define custom G-code blocks injected into the output.

Program Start
Inserted once at the top of the output file.
Program End
Inserted once at the bottom of the output file.
Stroke
Runs for every stroke.
Available Placeholders:
{X1} {Y1} {X2} {Y2} {Z_DOWN} {Z_UP} {FEED}
Color Change
Runs every time the color changes.
Available Placeholders:
{COLOR_INDEX} (1-based), {R} {G} {B} incoming color, {Z_UP}.
Dip Dip
Runs when a color with dipping enabled loads ink.
Available Placeholders:
{DIP_X} {DIP_Y} {DIP_Z} dip coords, {Z_UP} travel height, {FEED}.

Settings for color specific blocks - generate strokes for availability.
